# Examples

These examples contain code generated by OpenCLI along with minimal implementations required for the corresponding `ActionInterface`

## Cobra (Go)

```sh
# from repository root
ocli gen cli \
  --framework cobra \
  --out ./examples/code/cobra/pleasantries/internal \
  ./examples/pleasantries-cli.ocs.yaml
```

Files output to `examples/code/cobra/pleasantries`

## Yargs (Typescript)

```sh
# from repository root
ocli gen cli \
  --framework yargs \
  --out ./examples/code/yargs/pleasantries/src \
  ./examples/pleasantries-cli.ocs.yaml
```

Files output to `examples/code/yargs/pleasantries`
