gh alias set <alias> <expansion> [flags]
Arguments
  • <alias>
    The name of the alias that will expand into a full GitHub command
  • <expansion>
    The full github command that is being aliased
Flags
  • --clobber
    Overwrite existing aliases of the same name
  • -s, --shell
    Declare an alias to be passed through a shell interpreter
